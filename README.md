# Hormiga de Langton

## Descripción

Este proyecto es una simulación interactiva de la **Hormiga de Langton** desarrollada en **HTML, CSS y JavaScript** utilizando `<canvas>`.

La simulación muestra cómo reglas extremadamente simples pueden generar patrones complejos y comportamientos emergentes.

---

## Funcionamiento

La hormiga sigue estas reglas:

* Si está sobre una celda blanca:

  * gira a la derecha,
  * cambia la celda a negra,
  * avanza una posición.

* Si está sobre una celda negra:

  * gira a la izquierda,
  * cambia la celda a blanca,
  * avanza una posición.

---

## Cómo ejecutar el proyecto

1. Descarga el archivo `index.html`.
2. Ábrelo en cualquier navegador web moderno.
3. Presiona el botón **"Iniciar Simulación"**.
